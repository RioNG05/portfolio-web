# portfolio-web
Creating and Hosting a Responsive Portfolio Website
Objective
• Build a personal portfolio site using HTML, CSS, JavaScript, and Bootstrap.
• Apply responsive design principles for mobile, tablet, and desktop.
• Deploy the site using Netlify hosting.
• Include essential sections: Navbar, Header, Hero Section, Footer, Contact,
Home, About.

Sub Problems
1. HTML & Bootstrap Layout
o Create semantic structure with header, nav, main, section, footer.
o Use Bootstrap components (navbar, grid system, buttons, forms).
2. CSS Styling
o Customize Bootstrap with your own CSS (colors, fonts, spacing).
o Ensure consistent design across all sections.
3. JavaScript Functionality
o Add interactivity (e.g., smooth scroll, form validation, dynamic year in footer).

4. Responsive Design
o Implement mobile-first design.
o Use Bootstrap’s grid and media queries for responsiveness.
5. Deployment
o Host the portfolio on Netlify.
o Provide the live URL in your submission.
